# mindmaplib

Embeddable mindmap engine for web applications. Drop-in interactive mind maps in any portal, dashboard, or app.

## Why

Most mindmap tools are standalone apps. If you want a mindmap *inside* your product, you're stuck bolting a canvas library onto a layout engine and rebuilding pan/zoom/edit from scratch. mindmaplib is the engine layer: hand it data, get an interactive mind map widget back.

## Status

Early development. Private repo — will go open source (MIT) once the core engine is stable.
